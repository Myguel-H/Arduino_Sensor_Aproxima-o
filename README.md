Arduino - Sistema de Aproximação

Um projeto simples feito com Arduino que usa um sensor de proximidade para acionar um alarme quando um objeto é detectado.

-- Componentes necessários

1 Arduino Uno
1 Cabo USB
1 Sensor ultrassônico HC-SR04
1 Protoboard
Cabos jumper (macho-macho)
1 Buzzer ativo 5V
1 Motor / Atuador / Servo (depende do projeto)
Resistor (usado no LED da imagem opcionalmente)

-Windows

-Linux

  1. sudo apt install arduino
  2. arduino

-- Como compilar e enviar para o Arduino

  Abra o Arduino IDE
  Cole o código do projeto
  Vá em Ferramentas > Placa e selecione Arduino Uno
  Em Porta, selecione a porta onde seu Arduino foi detectado (ex: /dev/ttyACM0)
  No canto superior esquerdo, clique em 'Carregar' e ele ja ira fazer a conexão com o arduino.
